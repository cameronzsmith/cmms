▸ new Get(id, sql)

Gets a specific record from the supplied ID. Returns the data found from the SQL query if successful.

new Get(id: Number, sql: String): Object
Parameters
id (Number) The ID we wish to look up
sql (String) The SQL query to execute that's returned
Returns
Object: Returns the result of our SQL query from the supplied parameters.
▸ new GetAll(params, sql, table, alias)

Gets a paginated list of records based on the SQL query provided.

new GetAll(params: Object, sql: String, table: String, alias: String): Object
Parameters
params (Object) The parameter object from the HTTP request.
sql (String) The SQL query to execute that's returned.
table (String) The name of the table to get the record count from
alias (String = table+"s") The alias of the object that's returned in the returned result object
Returns
Object: Returns success status, and result object with user data, the amount of users, and pagination info.

▸ new CheckPermissions(session, groupsAllowed)

This function checks if the authenticated user has security access to the route they're requesting.

new CheckPermissions(session: Connection, groupsAllowed: Object): Object
Parameters
session (Connection) Connection object that has the authenticated user data.
groupsAllowed (Object) Array of Security Groups allowed to perform the request. Accepts: Administrator, Lead, Technician, Read Only
Returns
Object: Returns success status and the group the user belongs to. If a user doesn't belong to a group, it returns an error message instead of the security group.
